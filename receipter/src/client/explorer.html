How Walrus is used

Walrus is the durable evidence layer.

Full evidence bundles Worker claims, source observations, verification manifests, and delivery artifacts live as durable Walrus receipt blobs.
Portable passports Each worker passport rolls up Walrus-backed receipts so future buyers can route by prior verified deliveries.
Readback verification Receipter reads the blob back, recomputes hashes, and only then lets the receipt count toward clean reputation.
Reusable context Future buyers and agents can reuse prior Walrus receipts as audit context instead of trusting Receipter's private database.
